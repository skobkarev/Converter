Описание:

Конвертер валют на SpringBoot с применением:
 - Spring MVC
 - Thymeleaf
 - Spring Boot Security 
 - BootStrap

БД - MySQL


Вход в программу осуществляется через простую авторизацию. В файле SecurityConfig записаны два пользователя
Alex с паролем 123
и
Boss с паролем 456


Программа получает актуальный курс валют валют по ссылке http://www.cbr.ru/scripts/XML_daily.asp",
если в БД хранится урс валют не на текущую дату, то обновляет его до текущего, парсит до читаемого вида и выдает
в виде HTML страницы. 
Позволяет конвертировать одну валюту в другую и хранит историю конвертаций.

В папке 
ScriptsMySQL лежат два скрипта для MYSQL. Их необходимо скачать и запустить для создания таблиц.
